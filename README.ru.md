# Chrome Mistral OCR

Расширение Chrome для транскрибирования документов и изображений с помощью Mistral OCR. Оно работает из popup расширения, сохраняет результаты локально и позволяет просматривать, редактировать, копировать, экспортировать Markdown, экспортировать изображения и формировать PDF из отрендеренного предпросмотра.

## Конфиденциальность

Это расширение отправляет в Mistral AI документы, изображения или файлы, которые вы выбираете для обработки. Используя его, вы соглашаетесь с тем, что эти материалы будут обрабатываться в соответствии с текущими условиями и политиками Mistral AI.

Условия Mistral AI: https://mistral.ai/terms

Важные замечания:

- Это расширение является независимым и не связано с Mistral AI.
- Ваш API key хранится в `chrome.storage.local` внутри Chrome.
- Если вы обрабатываете чувствительные документы, сначала проверьте условия конфиденциальности вашего плана Mistral AI.

## Основные возможности

- OCR с `mistral-ocr-latest`.
- Два API key с настраиваемыми именами и быстрым переключателем в popup.
- Опциональное извлечение изображений.
- Экспорт изображений как ZIP или как отдельных файлов.
- Preview/Edit с Markdown-редактором, отрендеренным видом, опциональным автосохранением, копированием, экспортом Markdown и экспортом PDF.
- Подсветка OCR-слов с низкой уверенностью.
- Полный список сохранённых транскрипций с поиском по имени, URL, SHA или типу.
- Совместимость с Google Docs, Google Slides и Google Sheets.
- Автоматическая локализация по языку браузера с английским как fallback.

## Первый запуск

1. Создайте API key в Mistral Console: https://console.mistral.ai/
2. Откройте расширение и нажмите `Configure`.
3. Вставьте ключ в `API key 1`.
4. При необходимости настройте `API key 2`, чтобы переключаться между двумя аккаунтами или профилями.
5. Нажмите `Save`.
6. Откройте в Chrome совместимый документ или изображение.
7. Откройте расширение `Mistral OCR`.
8. При необходимости включите или выключите `Download images`.
9. Нажмите `Transcribe (OCR)`.

Если для текущего URL уже существует транскрипция, основная кнопка изменится на `Overwrite`.

Важно: чтобы использовать локальные PDF или файлы с `file://`, включите для этого расширения `Allow access to file URLs` в `chrome://extensions`.

## Поддерживаемые форматы

В основном поддерживает PDF-документы, которые отображаются Chrome в отрендеренном виде.

Google Docs, Google Slides и Google Sheets по умолчанию экспортируются как PDF. Нативные форматы Google (docx; pptx; xlsx; csv) также можно включить в Options. Нативные форматы могут повысить точность OCR, но они также могут раскрыть скрытый текст, содержащийся в исходном файле.

Кроме того, расширение может работать с любым изображением, которое Chrome отдельно рендерит в новой вкладке. Эти форматы включают: jpg, jpeg, png, avif, gif, bmp и webp.

## Устранение неполадок

Если OCR не запускается, убедитесь, что API key сохранён и что активная вкладка содержит совместимый ресурс. Если фоновая OCR-задача уже выполняется, дождитесь её завершения или отмените её.

Если локальные файлы не работают, откройте `chrome://extensions`, перейдите в сведения об этом расширении и включите `Allow access to file URLs`.
