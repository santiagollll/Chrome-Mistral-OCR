# Chrome Mistral OCR

إضافة Chrome لنسخ المستندات والصور باستخدام Mistral OCR. تعمل من خلال نافذة الإضافة المنبثقة، وتحفظ النتائج محليًا، وتسمح بالمعاينة والتحرير والنسخ وتصدير Markdown وتصدير الصور وإنشاء مخرجات PDF من المعاينة المولدة.

## الخصوصية

ترسل هذه الإضافة إلى Mistral AI المستندات أو الصور أو الملفات التي تختار معالجتها. باستخدامها، فإنك توافق على أن هذه المحتويات ستتم معالجتها وفقًا للشروط والسياسات الحالية الخاصة بـ Mistral AI.

شروط Mistral AI: https://mistral.ai/terms

ملاحظات مهمة:

- هذه الإضافة مستقلة وليست مرتبطة بـ Mistral AI.
- يتم حفظ API key داخل Chrome في `chrome.storage.local`.
- إذا كنت تعالج مستندات حساسة، فراجع أولًا شروط الخصوصية في خطة Mistral AI الخاصة بك.

## الميزات الرئيسية

- OCR باستخدام `mistral-ocr-latest`.
- مفتاحا API مع أسماء مخصصة ومبدل سريع داخل النافذة المنبثقة.
- استخراج الصور بشكل اختياري.
- تصدير الصور كملف ZIP أو كملفات منفصلة.
- Preview/Edit مع محرر Markdown وعرض مرئي وحفظ تلقائي اختياري ونسخ وتصدير Markdown وتصدير PDF.
- تمييز الكلمات ذات الثقة المنخفضة في OCR.
- قائمة كاملة بالنصوص المحفوظة مع البحث بالاسم أو URL أو SHA أو النوع.
- دعم Google Docs وGoogle Slides وGoogle Sheets.
- توطين تلقائي حسب لغة المتصفح مع الإنجليزية كلغة بديلة.

## الاستخدام الأول

1. أنشئ API key في Mistral Console: https://console.mistral.ai/
2. افتح الإضافة واضغط على `Configure`.
3. الصق المفتاح في `API key 1`.
4. يمكنك أيضًا إعداد `API key 2` للتبديل بين حسابين أو ملفين شخصيين.
5. اضغط `Save`.
6. افتح في Chrome مستندًا أو صورة متوافقة.
7. افتح إضافة `Mistral OCR`.
8. فعّل أو عطّل `Download images` حسب الحاجة.
9. اضغط `Transcribe (OCR)`.

إذا كانت هناك نسخة منسوخة مسبقًا لنفس URL الحالي، فسيتغير الزر الرئيسي إلى `Overwrite`.

مهم: لاستخدام ملفات PDF أو الملفات المحلية عبر `file://`، فعّل `Allow access to file URLs` لهذه الإضافة من `chrome://extensions`.

## الصيغ المدعومة

- PDF.
- الصور: `jpg`, `jpeg`, `png`, `avif`, `tif`, `tiff`, `gif`, `heic`, `heif`, `bmp`, `webp`.
- المستندات/الملفات: `docx`, `doc`, `pptx`, `ppt`, `xlsx`, `csv`, `txt`, `epub`, `xml`, `rtf`, `odt`, `bib`, `fb2`, `ipynb`, `tex`, `opml`, `man`.
- يتم تصدير Google Docs وSlides وSheets إلى PDF افتراضيًا، مع إمكانية استخدام الصيغ الأصلية اختياريًا من خلال Options.

يمكن أن تحسن صيغ Google الأصلية الدقة، لكنها قد تكشف أيضًا نصًا مخفيًا داخل الملف الأصلي.

## استكشاف الأخطاء وإصلاحها

إذا لم يبدأ OCR، فتحقق من أن API key محفوظة وأن علامة التبويب النشطة تحتوي على مورد متوافق. إذا كانت هناك مهمة OCR تعمل في الخلفية بالفعل، فانتظر حتى تنتهي أو قم بإلغائها.

إذا كانت الملفات المحلية لا تعمل، افتح `chrome://extensions` ثم افتح تفاصيل هذه الإضافة وفعّل `Allow access to file URLs`.
