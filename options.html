Primer uso

Conecta Mistral OCR con tu API key

La extensión funciona localmente en Chrome, pero el OCR se ejecuta usando tu cuenta de Mistral.

  1. Abre Mistral Console y crea una API key.
  2. Pega esa clave en API key 1.
  3. Haz click en Guardar.
  4. Abre un PDF, documento o imagen compatible y usa Transcribir (OCR) desde el popup.

Credencial opcional

API key 2

La segunda API key sirve para alternar rápido entre dos cuentas o perfiles de Mistral desde el popup.

  • El plan gratis de Mistral puede usar tus documentos para mejorar sus modelos. Eso significa menos privacidad.
  • El plan de pago de Mistral puede configurarse para no usar tus documentos para entrenamiento. Eso da más privacidad.
  • La idea práctica: usar gratis cuando no importa tanto la privacidad, y usar pago cuando el documento es sensible.
  • Si esta clave queda vacía, el popup no mostrará selector de API key.

Extracción y Google Workspace

Estas opciones usan extracción de header/footer de Mistral para ocultarlos del resultado OCR final.

.docx suele ser más preciso que PDF. Mistral lee mejor el texto real y las imágenes del documento.

  • Problema importante: Mistral también puede leer texto oculto.
  • Texto oculto significa texto que está en el archivo, pero tú no lo ves. Por ejemplo: letra tamaño 1, color blanco, escondida en una esquina.
  • A veces profesores ponen instrucciones ocultas para perjudicar el uso de IA. Si usas .docx, Mistral puede transcribir eso también.
  • Además, los .docx pueden pesar más y superar el límite de 50 MB de Mistral.

.pptx funciona parecido a .docx: puede ser muy preciso, pero Mistral también puede leer texto oculto.

  • Tiene el mismo riesgo de instrucciones ocultas que .docx.
  • Problema extra: las imágenes pueden quedar con tamaños raros o desajustados en la transcripción.
  • Úsalo sólo cuando necesitas mucha precisión y PDF no basta.
Cada servicio puede exportarse en su formato Office/CSV correspondiente en vez de PDF.
Ayuda .xlsx Ayuda .csv

.xlsx suele ser mejor que PDF para planillas. Mistral entiende mejor tablas y datos.

  • Es útil si lo importante son números, tablas o estructura de hoja de cálculo.
  • También puede incluir texto oculto del archivo original, igual que .docx.

.csv puede servir para datos, pero es el formato más delicado.

  • El problema: las tablas pueden quedar corridas. Si una columna se mueve, los datos pueden quedar mal.
  • Eso puede ser fatal si estás trabajando con notas, montos, fechas o datos importantes.
  • Úsalo sólo si ya sabes manejar CSV y vas a revisar el resultado.

Nota: si vas a abrir PDFs locales (file://), en chrome://extensions habilita “Permitir el acceso a URLs de archivos” para esta extensión.
